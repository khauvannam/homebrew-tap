# homebrew-tap

Official Homebrew tap for installing **statora**.

## Install

First add the tap:

```bash
brew tap khauvannam/tap
```

Then install **statora**:

```bash
brew install statora
```

## Upgrade

```bash
brew upgrade statora
```

## Uninstall

```bash
brew uninstall statora
```

## About statora

**statora** is a high-performance PHP runtime manager designed to manage:

* PHP versions
* Composer versions
* PHP extensions

Built with performance and developer productivity in mind.
